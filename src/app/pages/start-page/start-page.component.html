Instructions

  • To move use the keys "W", "A", "S", "D".
  • To shoot use the "SPACE" key.
  • To collect the gold use the "ENTER" key.
  • To win you must collect at least one gold and return to the starting square.
  • To kill the WUMPUS you must be on the square next to it and facing it.
  • The board is set randomly each game.
